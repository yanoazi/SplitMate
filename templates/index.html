Taiwan · LINE group ledger

SplitMate 在聊天裡記帳 在網頁裡結算

不是逼大家開網站的分帳 App。
付款人在 LINE 打一句 #分帳，對帳時再打開專屬頁面。

開啟 Demo 往下看故事
01 / Write

記帳發生在群組裡

聚餐結束不用切 App。付款人傳送： 

#分帳 900 火鍋 @小美 @小王

@ 後點選成員可綁 LINE ID；只打名字也能記，之後用 #合併 補綁。

02 / Read

結算發生在網頁上

最少轉帳

把交叉欠款壓成最少幾筆「誰付給誰」。

勾選相抵

只算你勾的那幾筆帳單，再批次標記已付。

PIN 編輯

群裡打 #網頁 取得 PIN，才能刪除或結清。

03 / Commands

群組指令很短

  • #分帳付款人記一筆
  • #結算最少轉帳結果
  • #欠款未付摘要
  • #成員🔗 / ❓
  • #合併補綁 LINE ID
  • #網頁連結 + PIN

完整說明在群裡打 #幫助。
